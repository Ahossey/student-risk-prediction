# Data

The raw dataset used in this project is publicly available from the 
UCI Machine Learning Repository.

## Download Instructions

1. Visit: https://archive.uci.edu/dataset/320/student+performance
2. Click Download
3. Extract the zip file
4. Place `student-mat.csv` and `student-por.csv` in `data/raw/`
5. Run notebooks in order starting from `01_data_acquisition.ipynb`

## Citation

Cortez, P., & Silva, A. (2008). Using data mining to predict 
secondary school student performance. FUBUTEC 2008.
